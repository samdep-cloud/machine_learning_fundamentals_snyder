# Machine Learning Fundamentals

Coursework from **Machine Learning Fundamentals** (Dr. Snyder, Boston University MS Data Science program).

This repository contains Jupyter notebooks covering core ML concepts and implementations including regression, classification, model evaluation, and practical applications of supervised learning techniques.

## Notebooks

- `homework_01.ipynb` - First steps with linear regression
- `homework_02.ipynb.ipynb` - Linear regression in theory and practice
- `homework_03.ipynb.ipynb` - Linear regression in theory and practice
- `homework_04.ipynb.ipynb` - Polynomial regression, etc.
- `homework_05.ipynb.ipynb` - Tools: Null values, categorical features and cross-validation
- `homework_06.ipynb.ipynb` - Feature selection and regularization on the Ames Housing dataset
- `homework_07.ipynb.ipynb` - Decision trees
- `homework_08.ipynb.ipynb` - Ensemble methods and gradient boosting
- `homework_09.ipynb.ipynb` - Classification
- `milestone_01.ipynb.ipynb` - EDA and pre-processing (group work)
- `milestone_02.ipynb.ipynb` - Feature engineering and modeling
- etc.

## Tools & Libraries
- Python, scikit-learn, pandas, numpy, matplotlib, etc.
